HTML language

HTML (HyperText Markup Language) is the standard language used to create and structure documents for the World Wide Web. It is not a programming language; rather, it is a markup language that uses a series of elements to "mark up" content so a web browser knows how to display it. 

HTML5 is the modern iteration of HyperText Markup Language.
    Unlike older HTML versions, HTML5 adds native multimedia support, semantic tags, and built-in web application APIs.
